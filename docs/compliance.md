# 合规声明（Compliance）

## 使用边界

本工具**仅限于用户自有账号**的自动化运营。典型合法场景：

- 管理自己闲置物品的发布与下架
- 自动回复买家咨询（结合 AI Agent）
- 定时刷新自己的商品

## 明确禁止

以下使用场景明确违反本工具的设计初衷，**作者不提供支持**：

- **刷单、虚假交易、炒作信用**：近年已有针对闲鱼刷单的司法判例
- **黑灰产批量铺货**：批量上架侵权/虚假/违禁商品
- **针对第三方的商业化 SaaS**：把本工具包装成服务卖给他人运营闲鱼
- **数据爬取贩卖**：批量抓取商品/用户数据用于商业分析
- **任何违反《闲鱼用户服务协议》《淘宝平台服务协议》的行为**

## 风控建议

内置护栏默认配置：
- 写操作限流：**1 次 / 分钟**（可通过 `GOOFISH_WRITE_RPM` 调，不建议调高）
- 风控熔断时长：**10 分钟**（可通过 `GOOFISH_CIRCUIT_BREAK_MINUTES` 调）

**强烈建议**：
- 单账号每日写操作（发布+下架+改价）不超过 **20 次**
- 不要多账号共享 IP
- 发布内容避免短时间内大量同质化（AI 识图可能命中）
- 触发风控后至少冷却 **30 分钟以上**再继续
