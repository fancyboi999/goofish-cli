# 标题与描述写作公式

## 标题：5 段式

```
[品牌] [核心词] [关键属性] [适用场景/对象] [情感/状态词]
```

### 示例

**数码**
- 原料：索尼 WH-1000XM5 降噪耳机 黑色 闲置 95 新
- 5 段式：`索尼 WH-1000XM5 降噪头戴式耳机 通勤午睡神器 95新原装盒`
  - 品牌：索尼
  - 核心词：WH-1000XM5 降噪头戴式耳机
  - 关键属性：（型号含在核心词里，省）
  - 场景：通勤午睡神器
  - 情感/状态：95新原装盒

**服饰**
- 原料：优衣库 女 M 码 羊毛大衣 黑色 有点起球
- 5 段式：`优衣库 羊毛双面呢大衣 女M码 秋冬通勤 轻微起球可议价`

**家居**
- 原料：宜家 台灯 白色 刚买 不太合适想出
- 5 段式：`宜家 RANARP 工作台灯 工业风 书房宿舍 全新带包装包邮`

### 长度约束

- 闲鱼标题上限 30 字符（中文 1 字算 1 字符）
- 最优区间 **18-28 字**（太短信息不足，太长截断）
- 关键词前置（品牌 + 核心词 必须在前 10 字内）

### 常见错误

| 不好 | 好 |
|---|---|
| "耳机出闲置" | "索尼 WH-1000XM5 降噪头戴式耳机 95 新" |
| "全网最低价 iPhone" | "iPhone 14 Pro 256G 深空黑 国行 9 成新" |
| "超好的大衣便宜出" | "优衣库 羊毛大衣 女M码 秋冬通勤 9成新" |

前者全是"情感+模糊词"——搜不到、没信任感。后者信息密度高。

### 违禁词快速避雷（详见 risk-guard）

- 不要：最 / 顶 / 第一 / 绝对 / 全网 / 独家 / 唯一
- 不要：亏本 / 跳楼价 / 白菜价（这些是"虚假促销"高危词）
- 不要：微信 / VX / 加我 / 联系我（外联红线）
- 不要：纯复制官方宣传语（"震撼登场"、"全新体验"）

## 描述：骨架

```
【商品】（品牌 + 型号 + 规格 + 购买渠道）
【成色】（X 成新 + 使用时长 + 是否原装附件）
【瑕疵】（有就写，拍特写图对应；没有就写"外观内功能均无明显问题"）
【为什么卖】（升级 / 不常用 / 搬家出闲置）
【价格】（是否议价；底线建议不写在描述里，藏在心里）
【发货】（发货地 + 快递 + 包邮与否 + 一般多久发）
【注意】（小刀政策 / 拒绝内容 / 售后说明）
```

### 示例

```
【商品】索尼 WH-1000XM5 降噪头戴式耳机，黑色，国行京东自营购入，2024.09 入手
【成色】使用约半年，95 新。头梁、耳罩、按键均正常。盒子、充电线、飞机头转接头、说明书齐全
【瑕疵】外观无磕碰；耳罩有一点使用痕迹但不影响佩戴（图 5）
【为什么卖】升级了新款 XM6，这个就闲置了
【价格】1500 出，小刀友好（5%以内随便砍），超过 1300 勿扰
【发货】上海同城可自提；外地顺丰到付或加 15 元包邮
【注意】接受 7 天无理由但请保护好外观和包装；耳机听力损伤类投诉不接受（二手非新品）
```

### 描述长度

- 最少 **30 字**（少于就进不了权重池）
- 最优 **200-400 字**
- 超过 600 字买家基本不看，但图里放得下细节图就没事

## 瑕疵描述的黄金法则

**主动说 > 被问到说 > 发货后被发现**。

- 能拍特写的瑕疵 → 拍特写图 + 描述里标"图 X"
- 难拍的瑕疵（气味、偶发卡顿、屏幕老化色差）→ 描述里写明
- 功能性瑕疵（充电慢、掉漆、按键回弹松）→ 必须写，不写等于卖家欺诈

## 发货说明的隐藏好处

写清楚发货地 + 快递 + 时长 → 买家不会反复问"几天发货"、"走什么快递"，节省客服时间。
