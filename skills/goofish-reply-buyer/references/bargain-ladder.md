# 议价阶梯策略

## 三档模型（闲鱼卖家圈约定俗成）

| 档位 | 砍价幅度 | 心态 | 策略 |
|---|---|---|---|
| **小刀** | 5-15% | 普通买家，顺手砍砍 | **友好接**，让 5-10% 成交 |
| **大刀** | 15-40% | 真想要但不确定行情 / 有其他选择 | **看商品**，好卖硬挺，难卖给中间 |
| **屠龙刀** | > 40% | 多半是刷屏 / 薅羊毛 / 刚入坑 | **礼貌拒**，不纠缠 |

## 前置：卖家要先告诉 Agent 两个价

Agent 在启动议价流程前，**必须向用户确认**：

```
【商品】挂价 XXX 元
【底线】您的底线是？（Agent 心里数，不告诉买家）
【小刀线】可以让多少？（通常 5-10%）
```

没有这两个数，**不要起草议价回复**。Agent 瞎给价 = 卖家亏钱。

## 话术模板

### 小刀（砍 5-15%）

**方式 A：直接让**（推荐用于挂价偏高、急出）
- 买家："1500 行吗"（挂价 1600）
- 话术：`1500 可以的，爽快点就这个价` / `1500 成交`

**方式 B：再让一步**（留价格缓冲）
- 买家："1400 可以吗"（挂价 1500）
- 话术：`少一点可以，1450 吧` / `1480 可以，再少就真没利润了`

### 大刀（砍 15-40%）

**方式 A：委婉拒 + 给中间价**
- 买家："1200 出吗"（挂价 1600）
- 话术：`1200 真没法卖，1400 可以的`

**方式 B：讲理由**
- 话术：`这个我当时也是 XXX 买的，1200 我亏得太多了，最多 1450`

**方式 C：如果商品不好卖**
- 可以接受或靠近买家价
- 话术：`看您诚意满满，1250 可以，不过只能发 XX 快递（不包邮）` ← 附加条件保利润

### 屠龙刀（砍 40%+ 或完全不合理）

**方式 A：一句拒，不解释**
- 买家："500 出吗"（挂价 1600）
- 话术：`谢谢关注哈，这个价没法卖，您再看看`

**方式 B：幽默化解**
- 话术：`哈哈咱们差得有点多，诚心要再聊`

**方式 C：直接沉默**（如果已经拒过一次，第二次屠龙刀）
- 不回复 / 礼貌结束对话
- `祝您早日淘到合适的～`

## 议价博弈小技巧

### 1. "先让买家出价"
模糊问价（"便宜点"、"最低多少"）→ 不要先说数字，让对方出：
- `您出个价我看看～`
- `看您诚意，方便说个数吗`

**为什么**：你先说 1500，对方会砍到 1200；让对方先说 1200，你有 1500 的起点。

### 2. "绑定附加价值"
让价时绑定条件，避免对方以为"挂价是虚高"：
- `1500 的话我帮您顺丰包邮`
- `1450 可以，送您一个原装耳机套`

### 3. "锚定原价"
描述里或回复里提一下原价：
- `当时 2500 买的，留凭证 XX，1500 出真心不亏您`

### 4. "设置离场时机"
顽固议价超过 3-4 轮 → 要么接要么结束：
- `这价格真做不了，您再看看其他家哈`

## 话术 DO / DON'T

| DO | DON'T |
|---|---|
| 语气随意，像朋友 | `尊敬的买家您好` |
| 给具体数字 | `差不多就行` / `您说个价吧` |
| 让步有节奏（不要一砍就到底） | 第一轮就说底线 |
| 必要时结束对话 | 被纠缠 5 轮还在磨 |
| 议价僵住时加附加条件 | 无条件让价 |

## 风控红线回避

议价中禁止出现的话：
- ❌ "加我微信聊"
- ❌ "支付宝转账给你优惠"
- ❌ "绕过闲鱼 XX"
- ❌ "保证 / 绝对 / 承诺 + 假一赔十"（没凭证的承诺 = 虚假宣传）
- ❌ "全网最低 / 最便宜"（绝对化词）

## 议价后如何留存 context

- 用户说"这个我接受 1450"后 → Agent 记录"本会话已确认 1450"
- 下次买家再砍 → Agent 知道底线已经到 1450，不再让步
- 结束后提醒用户："可以让他直接下单付款了"

## 不适合议价的场景

| 场景 | 处理 |
|---|---|
| 已是挂价最低（卖家说不议价） | 直接告诉买家"一口价" |
| 买家用屠龙刀刷屏多个商品 | 统一模板拒，不一对一 |
| 涉嫌刷单测试（新号 + 屠龙刀） | 不回更安全 |
| 买家只发"?"或表情包 | 回开放问题引导明确意图 |
