# 闲鱼搜索排序 / 推荐权重因子

注意：**以下是社区逆向 + 卖家长期观察得出的经验规则**，非官方披露。Agent 使用时要说明"这是经验推断"。

## 一级因子（决定性）

### 1. 账号闲气值
- < 60：所有商品权重基本废
- 60-80：搜索露出 -30% ~ -50%
- 80-95：正常
- > 95：在推荐流有轻微加权

### 2. 商品当前状态
- 是否违规过（降价改标、外联记录、举报）
- 是否在"限流"标记下（App 里"降权提醒"有时会显示）

### 3. 初始 24 小时互动率
- 曝光 → 点击率（CTR） > 3% 视为健康
- 点击 → 咨询率 > 5% 视为有效流量
- 咨询 → 成交率 （这个权重低，闲鱼不强制成交）

## 二级因子（优化项）

### 4. 标题质量
- 关键词覆盖广度（品牌 + 型号 + 属性 + 场景 + 情感）
- 核心词前置
- 长度 18-28 字
- 不含违禁词

### 5. 图片质量
- 首图 1:1
- 图片数 ≥ 3
- 实拍 > 网图（反向搜图会被扣分）
- 不含水印 / 联系方式

### 6. 描述质量
- 长度 ≥ 100 字（短描述权重低）
- 结构完整（商品 / 成色 / 瑕疵 / 发货 都提及）
- 不含违禁词

### 7. 类目匹配度
- 类目错放 → 直接 -30%
- 类目精确到二级子类 > 只到一级

### 8. 价格合理性
- 和同类同品相比 70%-130% 范围内最健康
- 过高：低点击率，拉低权重
- 过低：疑似钓鱼，可能进审核

## 三级因子（边际）

### 9. 发布时间
- 晚 7-9 点、午休 12-1 点发布的曝光初速度更快
- 凌晨发 → 曝光慢增长（但不影响最终权重）

### 10. 发布地 / 同城流量
- 同城买家对"本地"商品有加权展示
- 发货地精确到区，同城推荐率更高

### 11. 账号行为
- 每日登录
- 适当擦亮（每件 1 次/天）
- 买家消息回复率 > 80%
- 发货速度（> 7 天未发货扣权重）

### 12. 互动反馈
- 收藏数（非决定性，但有权重）
- 浏览时长（Stay time 长 = 用户有兴趣）

## 降权因子（扣分项）

### A. 频繁编辑
- 发布后 1 小时内改 3+ 次
- 标题改 > 5 次
- 价格改 > 3 次

### B. 降价改标
- **最重的扣分项**，直接 -50%~70%
- 24 小时内的组合最严重

### C. 同款连发
- 短时间发多件几乎相同商品
- 触发"职业卖家"判定

### D. 外联词历史
- 聊天中曾发外联词 → 账号记录
- 商品描述含外联 → 商品直接下架

### E. 多号同 IP
- 同 IP 下多个号被识别为关联
- 一号违规所有受影响

### F. 拒发货记录
- 买家付款后 7 天未发货 → 账号扣分
- 连续 3 次 → 严重降权

## 怎么用这些规则

Agent 在做诊断时：
1. 先问用户"闲气值多少"（让他去 App 看）
2. 用 `item_view` 拉详情对照上述因子
3. 按一级 → 二级 → 三级 → 降权因子顺序过
4. **不要一次给一个长 list**，挑最可能的 2-3 个告诉用户

## 权重恢复周期

| 降权事由 | 恢复时间 |
|---|---|
| 频繁编辑 | 7 天不动就恢复 |
| 降价改标 | 7-14 天 |
| 外联历史 | 30 天（期间零违规） |
| 账号闲气值 | 视违规类型，30-90 天 |
| 类目错放 | 立即（重发即可） |

## 反常识

- **"擦亮"权重增量很小**。不擦几天不会掉多少，三方工具宣传"自动擦亮涨流量"是夸大。
- **"收藏"和曝光相关性弱**。刷收藏 = 白刷 + 被判异常。
- **"挂价高"不一定降权**。高价小众品有固定受众，看价格合理性（相对同品），不是绝对高低。
- **"删掉重发"不是万能解**。频繁删发 = 风控素材。只在确定原商品被永久降权时才用。
