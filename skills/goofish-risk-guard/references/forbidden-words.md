## 二类：外联 / 诱导线下（封号红线）

**这部分在 `external-contact-keywords.md` 里另有详细版**，这里只放商品描述里常见的：

- "加微/加V/加好友/私聊" + "谈" / "聊" / "方便"
- "先付定金"（没有闲鱼担保保护）
- "线下交易" / "当面交易更便宜"（鼓励私下的话术）
- "走支付宝/转账/走微信"（绕过闲鱼担保）
- "X年老店，认准本人" + 引导加联系方式

```python
def scan(text: str) -> list[dict]:
    hits = []
    for category, words in FORBIDDEN_WORDS.items():
        for w in words:
            if w in text:
                hits.append({
                    "category": category,
                    "word": w,
                    "replacement": REPLACEMENTS.get(w),
                    "severity": "block" if category in BLOCK_CATEGORIES else "warn"
                })
    return hits

BLOCK_CATEGORIES = {"绝对化", "外联", "品牌侵权", "医疗夸大", "政治敏感", "违禁品"}
```
