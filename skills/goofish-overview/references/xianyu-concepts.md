# 闲鱼概念词典

给 Agent 的背景知识，让它在和用户对话时知道用户在说什么。**不是操作手册**——具体操作在各专项 skill。

## 账户信用类

### 闲气值
- 卖家信用分（0-100），闲鱼 App "我的" → "闲鱼信誉" 可查
- 分段限权（官方未完全公开阈值，社区经验）：
  - **< 80**：搜索曝光降 50% 起步；严重者鱼塘发帖也被过滤
  - **< 60**：发布频次受限，新建商品不进推荐
  - **< 40**：接近限制登录的边缘
- 影响闲气值的行为：违规、纠纷、仅退款占比、主动关闭订单、外联词命中

### 实人认证
- 人脸 + 身份证，未认证卖家单日发布量受限（社区反馈约 3-5 件封顶）
- AI 不代操作。只在卖家说"发不出去/今天超额了"时提示检查

## 流量与曝光类

### 流量池（推荐池）
- 每件新商品发布后有 **24 小时初始推荐窗口**
- 窗口内曝光/点击/收藏比决定是否进下一级池
- **错过就错过了**，没有二次初始流量
- 所以：发布时机、首 24 小时数据表现、首小时点击率都是关键

### 擦亮
- 每件宝贝每天可免费"擦亮" 1 次，等价于重新入池（权重弱于新发）
- 官方推荐时段：7 / 11 / 14 / 18 点（用户活跃高峰）
- **三方工具的主阵地**——闲管家、鱼小铺都做自动擦亮
- 本工具当前版本**不做自动擦亮**（有替代：编辑重发）

### 编辑重发
- 改标题/描述/价格后重发，部分场景等价于擦亮
- **坑**：先降价再改标题 → 触发"虚假促销"，流量腰斩
- 正确姿势：改标题不改价；或只改价不改标题

### 降权 / 限流
- 不是官方术语，但卖家圈公认：曝光断崖 + 搜索排不到 = 限流信号
- 常见归因（按社区排序）：类目错放 / 外联词历史命中 / 降价改标 / 与买家争吵 / 多号同 IP 关联 / 虚假成色描述被举报

## 合规类

### 15 种违规（官方《闲鱼社区管理规则》）
主要类目：
1. **信息发布类**：类目错放、虚假商品、图文不符、诱导线下
2. **交易安全类**：外联（微信/VX/威信/V信/+我/私聊等）、诱导担保外付款
3. **内容健康类**：绝对化用语（最好/顶级/第一/绝对/全网最低）、侵权品牌词、擦边
4. **信用类**：刷单、虚假评价、恶意议价、威胁报复
5. **物流类**：伪造物流、不发货、使用无痕发货

具体词表见 `goofish-risk-guard/references/forbidden-words.md`。

### 外联词（Agent 绝对红线）
- 微信 / V信 / VX / 威信 / 薇信 / +我 / 扣我 / 加我 / 私聊走
- 谐音变体：微❤ / v❤ / vx / 维❤ / 葳信 / 围信 / 维芯
- 电话号：13x/15x/17x/18x/19x 开头 11 位数字
- QQ 号：5-12 位纯数字
- **Agent 看到用户在起草消息里带这些词，必须先警告、替换为合规话术**

### 仅退款
- 平台机制：买家可申请"仅退款不退货"
- 判定偏买家（二手商品本身瑕疵多，平台倾向保护买家）
- 对卖家：聊天时提前说明瑕疵、留证据是唯一的防御手段

### RGV587 / x5sec
- 服务端风控熔断（返回码 RGV587 或 h5 返回 `FAIL_SYS_ILLEGAL_ACCESS`）
- 触发后需要 `x5sec` cookie 才能恢复调用
- `x5sec` 只能**从浏览器重导 cookie**拿到（带 x5sec/mtop_partitioned_detect）
- 本工具的 `auth_reset_guard` 只解**本地熔断**，不等于服务端解封

## 定价与议价类

### 小刀 / 大刀 / 屠龙刀（卖家圈黑话）
- **小刀**：砍价 5-15%（合理范围，卖家预期内）
- **大刀**：砍价 15-40%（要看品，贵价物件常见）
- **屠龙刀**：砍价 40%+ 或直接砍到远低于行情（多半是刷屏测试，建议忽略或礼貌拒绝）

Agent 回议价时的阶梯在 `goofish-reply-buyer/references/bargain-ladder.md`。

### 定金 / 邮费 / 运费险
- 闲鱼原生支持"担保交易"，买家付款到平台、确认收货后卖家收款
- 运费险：买家买；卖家看要不要提醒买家
- **不要线下担保**——任何"先加微信谈"都属违规路径

## 账号矩阵类

### 多号 / 分身
- 专业卖家 3-5 小号分散类目
- **同 IP 关联**（一机多号/一家多号）会被判定关联账号——一号违规全部受牵连
- **同设备指纹**（同一浏览器 profile）同理
- 本工具当前是**单 session**（v0.2.x），多号切换靠用户换 `GOOFISH_COOKIES_PATH` 手动隔离

### 饱和式封号（2023 热词）
- 卖家圈对"一次性批量封号"现象的俗称
- 触发：外联词历史命中 + 同 IP 多号 + 刷单嫌疑
- 恢复：申诉周期长（7-30 天），部分无法恢复
